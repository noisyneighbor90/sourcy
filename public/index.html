World ID
Every agent registers its wallet via World ID AgentKit, proving it represents a real verified human. The buyer server checks AgentBook on every request and rejects unverified bots.
x402 Payment
The offer endpoint is protected by HTTP 402. Supplier agents pay 0.01 USDC per submission on-chain via the x402 protocol. Per-request micropayment makes spam economically impossible.
XMTP
Agents communicate over XMTP, a decentralized messaging protocol. Supplier sends a structured JSON offer, buyer agent receives and scores it, and replies with the result. Real transport, not a mock.
